# Dian-plugin-qq-group-manage

QQ群组管理插件 — 功能全面的群管机器人

## 功能特性

### 权限管理
- 5级权限体系：主人 > 小主人 > 超管 > 群管理 > 普通
- 主人、小主人全局生效
- 超管按群设置
- 群管理自动同步QQ群管理员

### 系统设置
- 开机/关机 — 启用/停用机器人
- 闭嘴/说话 — 停止/恢复发言
- 专属模式 — 仅响应主人指令
- 查群状态 — 查看群配置
- 运行状态 — 查看系统运行信息

### 成员管理
- 踢/踢黑 — 踢出群成员
- 禁言/解禁 — 禁言指定成员
- 全体禁言/解禁 — 全群禁言
- 批量撤回 — 撤回指定成员最近消息

### 黑白名单
- 拉黑/删黑 — 黑名单管理
- 拉白/删白 — 白名单管理
- 退群自动拉黑

### 广告杀手
- 关键词检测 — 自动识别广告
- 三种惩罚：撤回、禁言、踢人
- 全局/按群配置

### 群内提示
- 新人欢迎 — 自定义欢迎消息
- 退群提示 — 自定义退群消息
- 支持变量：{at} {username} {group_name} {member_count}

### 发言统计
- 个人发言统计
- 今日/本周/本月排行榜
- 群体排行榜

### 名片系统
- 全员加马甲 — 统一前缀
- 改名提示 — 监控名片修改
- 锁定名片 — 禁止修改

### 定时任务
- 定时开机/关机
- 定时禁言/解禁
- 循环消息
- 定时消息
- 支持时间格式：8时30分、每天8时0分、星期一8时0分

### 图片模式
- 文字/图片回复切换
- 使用Puppeteer渲染

### WebUI
- 仪表盘 — 系统概览
- 主人管理 — 权限配置
- 群组管理 — 群设置
- 广告杀手 — 关键词配置
- 模板编辑 — HTML模板
- 群内提示 — 通知配置
- 发言统计 — 排行榜
- 名片系统 — 名片配置
- 定时任务 — 任务管理
- 使用文档 — 指令帮助

## 指令列表

### 快速入门
```
群管 帮助 — 查看所有指令
```

### 权限管理
```
加主人 @QQ — 添加主人
加小主人 @QQ — 添加小主人
加超管 @QQ — 添加本群超管
我的身份 — 查看权限
```

### 成员管理
```
踢 @QQ — 踢出成员
禁言 @QQ 10 — 禁言10分钟
解禁 @QQ — 解除禁言
全体禁言 — 全群禁言
```

### 黑白名单
```
拉黑 @QQ — 拉黑并踢出
拉白 @QQ — 加入白名单
查看黑名单 — 查看所有黑名单
```

### 广告杀手
```
开启广告杀手 — 开启检测
添加禁言关键词 充值,加群 — 添加关键词
```

### 群内提示
```
设置新人欢迎 欢迎{at}进群！ — 设置欢迎语
开启欢迎 — 开启欢迎消息
```

### 发言统计
```
我的发言 — 查看个人统计
今日发言 — 今日排行榜
```

### 名片系统
```
开启名片系统 — 开启
全员加马甲 萌宝● — 添加前缀
```

### 定时任务
```
开定时任务 — 开启
设置全体禁言 23时0分 — 定时禁言
加循环消息 30[=]记得喝水 — 循环消息
查看定时任务 — 查看所有任务
```

### 系统设置
```
开机/关机 — 启用/停用
闭嘴/说话 — 停止/恢复发言
运行状态 — 查看系统信息
```

### 图片模式
```
图片模式 开启 — 图片回复
图片模式 关闭 — 文字回复
```

## 安装

1. 将插件ZIP包上传到Dian管理界面
2. 或手动复制到 `plugins/` 目录
3. 重启Dian服务

## 配置

插件配置通过WebUI管理界面进行，支持：
- 按群独立配置
- 全局默认配置
- 实时生效

## 开发

```bash
# 安装依赖
npm install

# 开发模式
npm run dev:ui

# 构建
npm run build

# 打包
npm run pack
```

## 技术栈

- TypeScript
- React + Vite
- SQLite (PluginStore)
- Puppeteer (图片渲染)
- OneBot API
