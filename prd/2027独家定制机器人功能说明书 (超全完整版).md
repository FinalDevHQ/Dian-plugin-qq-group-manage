## 1. 问答系统
用于设置群内的自动回复，支持精确匹配和模糊匹配。

| 指令 | 解释说明 | 示范 |
| :--- | :--- | :--- |
| **精确问x请回答y** | 设置精确问答，支持图片、网址。多个问题用逗号隔开。 | `精确问A,B,C请回答y` |
| **模糊问x请回答y** | 设置模糊问答，只要包含关键词即触发。 | `模糊问x请回答y` |
| **精确/模糊问x请回答【卡片】** | 触发后机器人分享卡片消息（小程序、App分享等）。 | `精确问x请回答【卡片】` |
| **删除问答 [x]** | 删除指定的问答，系统自动识别精确或模糊。 | `删除问答 问题1` |
| **查看精确/模糊** | 查看当前群内所有的精确或模糊问答列表。 | `查看精确` |
| **清空精确/模糊** | 一键清空所有精确或模糊问答。 | `清空精确` |

**支持变量**：`【不要艾特】`、`【图片回复】`、`【@123】`（@指定QQ）。

## 2. 私聊控制 (高级管理)
通过私聊机器人实现远程管理操作（需超管权限）。

| 指令 | 解释说明 | 示范 |
| :--- | :--- | :--- |
| **查群列表** | 罗列机器人已激活的所有群。 | `查群列表` |
| **开群 [群号/全部]** | 开启新拉入的群或一键开启全部群。 | `开群 123456` |
| **关群 [群号]** | 停用指定群以释放名额。 | `关群 123456` |
| **换群 [旧群号] [新群号]** | 停用旧群并开启新群，自动复制设置。 | `换群 111 222` |
| **控制群 [群号]** | 进入私聊控制模式，后续指令将作用于该群。 | `控制群 123456` |
| **退出** | 退出控制群模式。 | `退出` |
| **转发 [内容]** | 将私聊消息转发给当前控制的群。 | `转发 大家好` |
| **群发公告 [内容]** | 向所有激活的群发布公告。 | `群发公告 维护通知` |
| **全体私聊 [内容]** | 私聊被控制群的所有成员（慎用）。 | `全体私聊 优惠活动` |

## 3. 发言奖励系统
通过发言概率或累计条数触发奖励。

| 指令 | 解释说明 | 示范 |
| :--- | :--- | :--- |
| **开启/关闭发言奖励** | 总开关。 | `开启发言奖励` |
| **设置发言概率 [数字]** | 设置千分比概率（如10表示1%）。 | `设置发言概率 10` |
| **设置发言奖励 [货币] [数量]** | 设置固定奖励。 | `设置发言奖励 金币 10` |
| **设置发言随机奖励 [货币] [低] [高]** | 设置随机奖励区间。 | `设置发言随机奖励 RMB 10 100` |
| **设置发言奖励条数 [数字]** | 累计发言达到指定条数时触发奖励。 | `设置发言奖励条数 100` |
| **设置每日触发次数 [数字]** | 限制每人每日获得奖励的次数。 | `设置每日触发次数 5` |

## 4. 定时任务
支持定时开关机、禁言、循环消息等。

| 指令 | 解释说明 | 示范 |
| :--- | :--- | :--- |
| **开启/关闭定时任务** | 总开关。 | `开启定时任务` |

| **加循环消息 [分钟] [=] [内容]** | 每隔指定分钟发送一次消息。 | `加循环消息 30[=]记得喝水` |
| **加定时消息 [日期] [=] [内容]** | 支持星期、日时分等格式。 | `加定时消息 星期一8时0分[=]周一好` |
